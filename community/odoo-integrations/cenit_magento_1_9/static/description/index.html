Magento_1_9 Integration

Powered by Cenit IO

Magento is an open source (opensource) platform, with which you can carry out all kinds of projects related to the sale on the Internet. This Cenit's module allows the integration of this platform with Odoo so that you can update the content types of Odoo in Magento.

Cenit IO

This addon needs the Cenit Itegrations Client (don't worry: try it for free)

Setting up...

First things first: Sign up with Magento_1_9. You must create an application to selecting REST Oauth consumers from the web service menu, enter the data of the name, key, secret and the callback url. These will be used for the authentication on cenit.
Magento's Dashboard
Magento Aplication Credentials

In Cenit you must access the authorization form of the magento collection, update the key and secret given by magento. Then when these parameters are ready, you must access the authorization menu and re-authorize.

Credentials on Cenit
Cenit's Authorize

You're now ready to work on Odoo module! To follow the steps above, see Cenit IO | Magento Cenit
