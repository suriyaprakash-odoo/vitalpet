Successfully manage your Product Category Company Parameters

When this module gets installed, it will create sub-menu named Account Properties under the Action menu in Product Categories.

The Action menu can be viewed only in the list view and when the product categories are selected.

When product category(s) is/are selected and the ‘Account Properties’, a wizard gets opened, where the Income and Expense needs to filled. Once filled and when button ‘Apply’ is clicked, the income and expense account gets applied for all the selected product category(s).

Note: It works only for the selected product categories.

This feature is done for multi-company environment where it works based in the company domain selected.
