Description

Conditional questions and file uploads

Conditional Questions

Setup questions which only appear when a certain option is selected from a previous question.

Instructions
1. Go to any survey question
2. Go into the conditional tab
3. Select a 'Multiple choice: only one answer' question from your survey
4. Select an option from that question
5. When a person is doing the survey and they select that option the conditional question will appear
6. e.g. person selects 'yes' to question 'do you like cars?' and the condition question 'what type of car?' appears

File uploads

Users can upload files during a survey.

Instructions
1. Add a new question to your survey
2. Select the new binary question type
3. When a user fills out the survey they will get to attach a file for that question

Survey Lead Funnel

At the end of a survey the user is able to enter thier name and email and they will become a lead with the newly completed lead associated to them.

Instructions
1. Go to a survey
2. Go into the options tab
3. Tick the lead survey checkbox
4. After the user is finished with the surbvey they will be presented with a form.
5. When the user fills in the form they will become a lead and the lead conversion process can begin
6. You can view the lead that did the survey by going into an individual survey answer
